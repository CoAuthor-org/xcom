# Sample Notes

## What is an OS?
An operating system manages hardware and software resources. It provides a stable way for applications to use the CPU, memory, and storage.

## Key concepts
- **Kernel**: Core of the OS; runs with highest privileges.
- **Process**: A program in execution. Has its own address space.
- **Thread**: Unit of execution within a process. Shares the process address space.
